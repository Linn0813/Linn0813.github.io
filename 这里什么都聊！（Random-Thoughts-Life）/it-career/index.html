жҜ•дёҡеҗҺз¬¬дёҖд»Ҫе·ҘдҪңжҖҺд№ҲйҖүпјҹеүҚз«ҜпјҹеҗҺз«ҜпјҹжөӢиҜ•пјҹз®—жі•пјҹдә§е“Ғпјҹ
йҖүеҜ№дәҶпјҢжҲҗй•ҝеҝ«гҖҒе·ҘдҪңејҖеҝғпјӣйҖүй”ҷдәҶпјҢеҸҜиғҪдёҖе№ҙеҗҺе°ұжғіиҪ¬иЎҢгҖӮрҹӨҜ

жң¬жҢҮеҚ—д»Һ е·ҘдҪңеҶ…е®№гҖҒжҠҖиғҪиҰҒжұӮгҖҒеҸ‘еұ•еүҚжҷҜгҖҒйҖӮеҗҲдәәзҫӨ зӯүи§’еәҰж·ұе…ҘеҜ№жҜ”пјҢеҠ©дҪ еҒҡеҮәжңҖдҪіеҶізӯ–пјҒ

рҹ’Ў е…ҲеҒҡдёӘжөӢиҜ•пјҢзңӢзңӢдҪ йҖӮеҗҲе“ӘдёӘж–№еҗ‘пјҹ

вң… дҪ жҳҜеҗҰе–ңж¬ўи°ғиҜ• UI з»„д»¶пјҢи°ғж•ҙж ·ејҸпјҹ вҶ’ еүҚз«Ҝ
вң… дҪ е–ңж¬ўдјҳеҢ–д»Јз ҒйҖ»иҫ‘пјҢз ”з©¶ж•°жҚ®еә“гҖҒAPIпјҹ вҶ’ еҗҺз«Ҝ
вң… дҪ ж“…й•ҝеҸ‘зҺ°й—®йўҳпјҢе–ңж¬ўжүҫ BugгҖҒжөӢиҜ•зі»з»ҹпјҹ вҶ’ жөӢиҜ•
вң… дҪ еҜ№ж•°еӯҰгҖҒжңәеҷЁеӯҰд№ гҖҒAI з ”з©¶ж„ҹе…ҙи¶Јпјҹ вҶ’ з®—жі•
вң… дҪ е–ңж¬ўжҖқиҖғдә§е“ҒеҰӮдҪ•и®ҫи®ЎгҖҒеҰӮдҪ•ж»Ўи¶із”ЁжҲ·йңҖжұӮпјҹ вҶ’ дә§е“Ғ

рҹҡҖ з»§з»ӯеҫҖдёӢзңӢпјҢдәҶи§Јеҗ„еІ—дҪҚзҡ„иҜҰз»Ҷи§ЈжһҗпјҒ

рҹ’» еүҚз«ҜејҖеҸ‘пјҡжү“йҖ з”ЁжҲ·з•Ңйқўзҡ„йӯ”жі•еёҲ

е…ій”®иҜҚпјҡз•ҢйқўгҖҒдәӨдә’гҖҒеҠЁз”»гҖҒз”ЁжҲ·дҪ“йӘҢ

вң… йҖӮеҗҲдҪ еҗ—пјҹ

  • рҹҺЁ е…іжіЁз”ЁжҲ·дҪ“йӘҢпјҢе–ңж¬ўйЎөйқўи®ҫи®ЎгҖҒеҠЁз”»е’ҢдәӨдә’дјҳеҢ–
  • рҹ–ҘпёҸ д»Јз Ғи§Ғж•Ҳеҝ«пјҢи°ғж•ҙж ·ејҸгҖҒж”№еҠЁ UIпјҢз«Ӣй©¬иғҪзңӢеҲ°ж•Ҳжһң
  • рҹ§  йҖ»иҫ‘иғҪеҠӣеҸҜд»ҘдёҖиҲ¬пјҢдҪҶиҰҒжңү е®ЎзҫҺ & дәӨдә’ж•Ҹж„ҹеәҰ

рҹҺҜ дё»иҰҒе·ҘдҪңеҶ…е®№

  • дҪҝз”Ё HTML / CSS / JavaScript жҗӯе»әзҪ‘йЎө
  • з ”з©¶еҗ„з§ҚеүҚз«ҜжЎҶжһ¶пјҲVueгҖҒReactгҖҒAngularпјү
  • еӨ„зҗҶйЎөйқўйҖӮй…ҚгҖҒеҠЁз”»дјҳеҢ–гҖҒз”ЁжҲ·дәӨдә’дҪ“йӘҢ
  • дёҺеҗҺз«ҜеҜ№жҺҘ APIпјҢи®©ж•°жҚ®еҠЁжҖҒеұ•зӨә

рҹ“Ҳ еҸ‘еұ•еүҚжҷҜ

  • Web еүҚз«Ҝе·ҘзЁӢеёҲпјҲVue / React ејҖеҸ‘пјү
  • з§»еҠЁз«ҜејҖеҸ‘пјҲFlutter / React Nativeпјү
  • еүҚз«Ҝжһ¶жһ„еёҲпјҲжҖ§иғҪдјҳеҢ–гҖҒеӨ§еһӢйЎ№зӣ®жһ¶жһ„пјү
  • е…Ёж ҲејҖеҸ‘пјҲеүҚз«Ҝ+еҗҺз«ҜпјҢйҖӮеҗҲеҲӣдёҡе’Ңе°Ҹеӣўйҳҹпјү

рҹҡЁ йҳІеқ‘жҢҮеҚ—
вқҢ иҜҜеҢәпјҡ д»ҘдёәеүҚз«ҜеҸӘеҶҷйЎөйқўпјҢе…¶е®һ ж•°жҚ®з®ЎзҗҶгҖҒжҖ§иғҪдјҳеҢ–гҖҒе·ҘзЁӢеҢ– д№ҹжҳҜйҮҚзӮ№пјҒ

рҹ”ҷ еҗҺз«ҜејҖеҸ‘пјҡйҖ»иҫ‘зјңеҜҶзҡ„е№•еҗҺж“ҚзӣҳжүӢ

е…ій”®иҜҚпјҡж•°жҚ®еә“гҖҒжңҚеҠЎеҷЁгҖҒдёҡеҠЎйҖ»иҫ‘гҖҒжҖ§иғҪдјҳеҢ–

вң… йҖӮеҗҲдҪ еҗ—пјҹ

  • вҡҷпёҸ йҖ»иҫ‘жҖқз»ҙејәпјҢе–ңж¬ўз ”з©¶ж•°жҚ®жөҒиҪ¬гҖҒзі»з»ҹжһ¶жһ„
  • рҹ”§ дёҚеңЁж„Ҹ UI и®ҫи®ЎпјҢжӣҙе…іжіЁ жҖ§иғҪдјҳеҢ– & д»Јз Ғжһ¶жһ„
  • рҹӣ пёҸ Debug дёҖж•ҙеӨ©дёҚи§үеҫ—зғҰпјҢзңӢеҲ°ж•°жҚ®жҲҗеҠҹиҝ”еӣһеҫҲжңүжҲҗе°ұж„ҹ

рҹҺҜ дё»иҰҒе·ҘдҪңеҶ…е®№

  • и®ҫи®Ў ж•°жҚ®еә“пјҢз®ЎзҗҶж•°жҚ®еӯҳеӮЁпјҲSQL / NoSQLпјү
  • зј–еҶҷ APIпјҢеӨ„зҗҶдёҡеҠЎйҖ»иҫ‘пјҢдёҺеүҚз«ҜеҜ№жҺҘ
  • еӨ„зҗҶ й«ҳе№¶еҸ‘гҖҒеҲҶеёғејҸжһ¶жһ„гҖҒзј“еӯҳдјҳеҢ–
  • з ”з©¶ дә‘и®Ўз®—гҖҒеӨ§ж•°жҚ®гҖҒеҫ®жңҚеҠЎжһ¶жһ„

рҹ“Ҳ еҸ‘еұ•еүҚжҷҜ

  • еҗҺз«ҜејҖеҸ‘е·ҘзЁӢеёҲпјҲJava / Python / Go / Node.jsпјү
  • ж•°жҚ®еә“з®ЎзҗҶе‘ҳпјҲDBAпјүпјҲSQLгҖҒж•°жҚ®еә“дјҳеҢ–пјү
  • еҗҺз«Ҝжһ¶жһ„еёҲпјҲеҫ®жңҚеҠЎгҖҒеӨ§ж•°жҚ®гҖҒеҲҶеёғејҸжһ¶жһ„пјү
  • дә‘и®Ўз®— & AI ж–№еҗ‘пјҲз»“еҗҲ AIгҖҒеӨ§ж•°жҚ®ејҖеҸ‘жҷәиғҪзі»з»ҹпјү

рҹҡЁ йҳІеқ‘жҢҮеҚ—
вқҢ иҜҜеҢәпјҡ д»ҘдёәеҗҺз«ҜеҸӘеҶҷ APIпјҢе…¶е®һ жһ¶жһ„и®ҫи®ЎгҖҒж•°жҚ®е®үе…ЁгҖҒжҖ§иғҪдјҳеҢ– д№ҹжҳҜж ёеҝғпјҒ

рҹ§җ жөӢиҜ•ејҖеҸ‘пјҡдә§е“ҒиҙЁйҮҸзҡ„е®ҲжҠӨиҖ…

е…ій”®иҜҚпјҡиҮӘеҠЁеҢ–гҖҒжҖ§иғҪгҖҒе®үе…ЁгҖҒиҙЁйҮҸдҝқйҡң

вң… йҖӮеҗҲдҪ еҗ—пјҹ

  • рҹ”Қ з»ҶеҝғпјҢе–ңж¬ўжүҫ BugгҖҒжӢҶи§Јй—®йўҳ
  • рҹӣ пёҸ д»Јз ҒиғҪеҠӣдёҚдёҖе®ҡиҰҒејәпјҢдҪҶиҰҒжңүиүҜеҘҪзҡ„ жөӢиҜ•жҖқз»ҙ
  • рҹ§җ ж„ҝж„ҸжҢ‘жҲҳеӨҚжқӮжөӢиҜ•еңәжҷҜпјҢжҸҗеҚҮдә§е“ҒиҙЁйҮҸ

рҹҺҜ дё»иҰҒе·ҘдҪңеҶ…е®№

  • и®ҫи®Ў жөӢиҜ•з”ЁдҫӢпјҢжүӢе·ҘжөӢиҜ•еҠҹиғҪпјҢжүҫ Bug
  • иҝӣиЎҢ жҺҘеҸЈиҮӘеҠЁеҢ–гҖҒUI иҮӘеҠЁеҢ–гҖҒжҖ§иғҪжөӢиҜ•
  • з ”з©¶еҰӮдҪ•и®©жөӢиҜ•иҮӘеҠЁеҢ–пјҢжҸҗй«ҳжөӢиҜ•ж•ҲзҺҮ
  • еҲҶжһҗж—Ҙеҝ—пјҢеӨҚзҺ° BugпјҢе’ҢејҖеҸ‘дёҖиө·и§ЈеҶій—®йўҳ

рҹ“Ҳ еҸ‘еұ•еүҚжҷҜ

  • иҮӘеҠЁеҢ–жөӢиҜ•пјҲSelenium / Appium / pytestпјү
  • жҖ§иғҪжөӢиҜ•пјҲJMeter / LoadRunnerпјү
  • е®үе…ЁжөӢиҜ•пјҲжё—йҖҸжөӢиҜ• / зҷҪеёҪй»‘е®ўпјү
  • жөӢиҜ•жһ¶жһ„еёҲпјҲиҙҹиҙЈдјҒдёҡзә§жөӢиҜ•зі»з»ҹи®ҫи®Ўпјү

рҹҡЁ йҳІеқ‘жҢҮеҚ—
вқҢ иҜҜеҢәпјҡ д»ҘдёәжөӢиҜ•еҸӘдјҡвҖңзӮ№зӮ№зӮ№вҖқпјҢе…¶е®һ иҮӘеҠЁеҢ–гҖҒжҖ§иғҪгҖҒе®үе…ЁжөӢиҜ• йғҪжҳҜжҠҖжңҜеҗ«йҮҸжһҒй«ҳзҡ„йўҶеҹҹпјҒ

рҹӨ– з®—жі•е·ҘзЁӢеёҲпјҡдәәе·ҘжҷәиғҪзҡ„еҲӣйҖ иҖ…

е…ій”®иҜҚпјҡAIгҖҒжңәеҷЁеӯҰд№ гҖҒеӨ§ж•°жҚ®гҖҒж·ұеәҰеӯҰд№ 

вң… йҖӮеҗҲдҪ еҗ—пјҹ

  • рҹӨ– зғӯзҲұж•°еӯҰпјҢеҜ№ ж•°жҚ®гҖҒAIгҖҒжЁЎеһӢи®ӯз»ғ жңүе…ҙи¶Ј
  • рҹ“Ҡ е–ңж¬ўз ”з©¶ жңәеҷЁеӯҰд№ гҖҒж·ұеәҰеӯҰд№ гҖҒж•°еӯҰе»әжЁЎ
  • рҹ”ў йҖ»иҫ‘иғҪеҠӣејәпјҢж„ҝж„Ҹй’»з ”еӨҚжқӮз®—жі•

рҹҺҜ дё»иҰҒе·ҘдҪңеҶ…е®№

  • з ”з©¶е’ҢдјҳеҢ– жңәеҷЁеӯҰд№  / ж·ұеәҰеӯҰд№ жЁЎеһӢ
  • еӨ„зҗҶ еӨ§и§„жЁЎж•°жҚ®пјҢжҸҗеҚҮз®—жі•зІҫеәҰ
  • з ”з©¶ и®Ўз®—жңәи§Ҷи§үгҖҒиҮӘз„¶иҜӯиЁҖеӨ„зҗҶпјҲNLPпјү
  • еҸӮдёҺ AI и®ӯз»ғпјҢи°ғдјҳжЁЎеһӢ

рҹ“Ҳ еҸ‘еұ•еүҚжҷҜ

  • жңәеҷЁеӯҰд№ е·ҘзЁӢеёҲпјҲAI жЁЎеһӢи®ӯз»ғгҖҒдјҳеҢ–пјү
  • и®Ўз®—жңәи§Ҷи§ү / NLP з ”з©¶е‘ҳпјҲеӣҫеғҸгҖҒиҜӯйҹігҖҒж–Үжң¬еӨ„зҗҶпјү
  • ж•°жҚ®з§‘еӯҰе®¶пјҲе•Ҷдёҡж•°жҚ®еҲҶжһҗгҖҒж•°жҚ®жҢ–жҺҳпјү

рҹҡЁ йҳІеқ‘жҢҮеҚ—
вқҢ иҜҜеҢәпјҡ д»Ҙдёәз®—жі• = и°ғеҢ…пјҢе…¶е®һ ж•°еӯҰеҠҹеә•гҖҒд»Јз ҒиғҪеҠӣгҖҒж•°жҚ®жё…жҙ— д№ҹеҫҲйҮҚиҰҒпјҒ

рҹ“Ң дә§е“Ғз»ҸзҗҶпјҡиҝһжҺҘжҠҖжңҜдёҺз”ЁжҲ·зҡ„жЎҘжўҒ

е…ій”®иҜҚпјҡз”ЁжҲ·йңҖжұӮгҖҒдә§е“Ғи®ҫи®ЎгҖҒе•ҶдёҡйҖ»иҫ‘гҖҒеёӮеңәеҲҶжһҗ

вң… йҖӮеҗҲдҪ еҗ—пјҹ

  • рҹҺҜ е…іжіЁз”ЁжҲ·дҪ“йӘҢпјҢе–ңж¬ўз ”з©¶дә§е“ҒеҰӮдҪ•дјҳеҢ–
  • рҹ“Ҡ йҖ»иҫ‘жё…жҷ°пјҢиғҪжҠҠеӨҚжқӮй—®йўҳжӢҶи§ЈжҲҗ жё…жҷ°зҡ„еҠҹиғҪ
  • рҹӨқ жІҹйҖҡиғҪеҠӣејәпјҢиғҪеҚҸи°ғжҠҖжңҜгҖҒи®ҫи®ЎгҖҒеёӮеңәзӯүеӣўйҳҹ

рҹҺҜ дё»иҰҒе·ҘдҪңеҶ…е®№

  • йңҖжұӮеҲҶжһҗпјҡз ”з©¶з”ЁжҲ·йңҖжұӮпјҢеҲ¶е®ҡдә§е“Ғж–№жЎҲ
  • дә§е“Ғи®ҫи®Ўпјҡз”»еҺҹеһӢеӣҫпјҲAxureгҖҒFigmaпјүгҖҒи®ҫи®ЎдәӨдә’йҖ»иҫ‘
  • йЎ№зӣ®з®ЎзҗҶпјҡжҺЁеҠЁејҖеҸ‘гҖҒжөӢиҜ•дёҠзәҝпјҢзЎ®дҝқдә§е“ҒиҗҪең°
  • ж•°жҚ®еҲҶжһҗпјҡи·ҹиёӘз”ЁжҲ·ж•°жҚ®пјҢдјҳеҢ–дә§е“Ғзӯ–з•Ҙ

рҹ“Ҳ еҸ‘еұ•еүҚжҷҜ

  • дә§е“Ғз»ҸзҗҶпјҲAPP / SaaS / жёёжҲҸ / зЎ¬д»¶зӯүж–№еҗ‘пјү
  • й«ҳзә§дә§е“Ғз»ҸзҗҶпјҲиҙҹиҙЈж ёеҝғдә§е“Ғзӯ–з•Ҙпјү
  • дә§е“ҒжҖ»зӣ‘ / CEOпјҲжӣҙй«ҳеұӮзҡ„дә§е“ҒеҶізӯ–пјү
