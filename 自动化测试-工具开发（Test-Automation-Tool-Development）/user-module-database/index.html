дҪҝз”Ё MySQL жҗӯе»әзҷ»еҪ•жіЁеҶҢжЁЎеқ—ж•°жҚ®еә“ рҹӣўпёҸ

еңЁеҪ“д»Ҡж•°еӯ—еҢ–зҡ„ж—¶д»ЈпјҢзҷ»еҪ•жіЁеҶҢеҠҹиғҪжҳҜеҗ„зұ»еә”з”ЁзЁӢеәҸдёҚеҸҜжҲ–зјәзҡ„дёҖйғЁеҲҶгҖӮиҖҢж•°жҚ®еә“дҪңдёәеӯҳеӮЁз”ЁжҲ·дҝЎжҒҜзҡ„ж ёеҝғпјҢе…¶жҗӯе»әзҡ„йҮҚиҰҒжҖ§дёҚиЁҖиҖҢе–»гҖӮд»ҠеӨ©пјҢжҲ‘д»¬е°ұжқҘиҜҰз»ҶжҺўи®ЁеҰӮдҪ•дҪҝз”Ё MySQL жҗӯе»әдёҖдёӘз”ЁдәҺзҷ»еҪ•жіЁеҶҢжЁЎеқ—зҡ„ж•°жҚ®еә“рҹҡҖгҖӮ

дёҖгҖҒзҺҜеўғеҮҶеӨҮ рҹӣ пёҸ

еңЁејҖе§Ӣжҗӯе»әж•°жҚ®еә“д№ӢеүҚпјҢжҲ‘д»¬йҰ–е…ҲиҰҒзЎ®дҝқе·Із»Ҹе®үиЈ…дәҶ MySQL ж•°жҚ®еә“пјҢе№¶дё”е®ғиғҪеӨҹжӯЈеёёеҗҜеҠЁе’Ңи®ҝй—®гҖӮиҝҷе°ұеҘҪжҜ”жҲ‘д»¬иҰҒзӣ–жҲҝеӯҗпјҢеҫ—е…ҲеҮҶеӨҮеҘҪе»әзӯ‘жқҗж–ҷдёҖж ·рҹ§ұгҖӮеҰӮжһңдҪ иҝҳжІЎжңүе®үиЈ… MySQLпјҢеҸҜд»Ҙд»Һе®ҳж–№зҪ‘з«ҷдёӢиҪҪйҖӮеҗҲдҪ ж“ҚдҪңзі»з»ҹзҡ„зүҲжң¬иҝӣиЎҢе®үиЈ…гҖӮе®үиЈ…е®ҢжҲҗеҗҺпјҢйҖҡиҝҮд»ҘдёӢе‘Ҫд»ӨжЈҖжҹҘ MySQL жҳҜеҗҰжӯЈеёёиҝҗиЎҢпјҡ

иҫ“е…ҘдҪ зҡ„еҜҶз ҒпјҢеҰӮжһңиғҪжҲҗеҠҹзҷ»еҪ•еҲ° MySQL е‘Ҫд»ӨиЎҢз•ҢйқўпјҢиҜҙжҳҺзҺҜеўғеҮҶеӨҮе°ұз»Әрҹ‘ҚгҖӮ

дәҢгҖҒеҲӣе»әж•°жҚ®еә“ рҹ“Ұ

жҺҘдёӢжқҘпјҢжҲ‘д»¬йңҖиҰҒеҲӣе»әдёҖдёӘдё“й—Ёз”ЁдәҺеӯҳеӮЁзҷ»еҪ•жіЁеҶҢзӣёе…іж•°жҚ®зҡ„ж•°жҚ®еә“гҖӮеҸҜд»ҘдҪҝз”Ёд»ҘдёӢ SQL иҜӯеҸҘеңЁ MySQL дёӯеҲӣе»әж•°жҚ®еә“пјҡ

жү§иЎҢдёҠиҝ°иҜӯеҸҘеҗҺпјҢMySQL дјҡеҲӣе»әдёҖдёӘеҗҚдёә test_platform зҡ„ж•°жҚ®еә“гҖӮдёәдәҶеҗҺз»ӯж“ҚдҪңж–№дҫҝпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨеҲҮжҚўеҲ°иҜҘж•°жҚ®еә“пјҡ

иҝҷж ·пјҢжҲ‘д»¬е°ұиҝӣе…ҘдәҶ test_platform ж•°жҚ®еә“зҡ„ж“ҚдҪңзҺҜеўғе•ҰрҹҺүгҖӮ

дёүгҖҒеҲӣе»әз”ЁжҲ·иЎЁ рҹ“Ӣ

еңЁ test_platform ж•°жҚ®еә“дёӯпјҢжҲ‘д»¬иҰҒеҲӣе»әдёҖдёӘз”ЁжҲ·иЎЁпјҲusersпјүпјҢз”ЁдәҺеӯҳеӮЁз”ЁжҲ·зҡ„зҷ»еҪ•жіЁеҶҢдҝЎжҒҜгҖӮз”ЁжҲ·иЎЁеә”еҢ…еҗ«з”ЁжҲ· IDгҖҒз”ЁжҲ·еҗҚгҖҒеҜҶз ҒгҖҒз”ЁжҲ·и§’иүІзӯүеӯ—ж®өгҖӮд»ҘдёӢжҳҜеҲӣе»әз”ЁжҲ·иЎЁзҡ„ SQL иҜӯеҸҘпјҡ

дёҠиҝ° SQL иҜӯеҸҘеҲӣе»әдәҶдёҖдёӘеҗҚдёә users зҡ„иЎЁпјҢеҗ„еӯ—ж®өиҜҙжҳҺеҰӮдёӢрҹ‘Үпјҡ

  • idпјҡз”ЁжҲ·зҡ„е”ҜдёҖж ҮиҜҶпјҢдҪҝз”ЁиҮӘеўһж•ҙж•°дҪңдёәдё»й”®гҖӮе°ұеғҸжҜҸдёӘдәәйғҪжңүдёҖдёӘзӢ¬дёҖж— дәҢзҡ„иә«д»ҪиҜҒеҸ·з ҒдёҖж ·пјҢid еҸҜд»ҘзЎ®дҝқжҜҸдёӘз”ЁжҲ·еңЁж•°жҚ®еә“дёӯзҡ„е”ҜдёҖжҖ§гҖӮ
  • usernameпјҡз”ЁжҲ·зҡ„зҷ»еҪ•еҗҚпјҢдёҚиғҪдёәз©әдё”еҝ…йЎ»е”ҜдёҖгҖӮиҝҷжҳҜз”ЁжҲ·зҷ»еҪ•ж—¶дҪҝз”Ёзҡ„еҗҚз§°пјҢиҰҒдҝқиҜҒе…¶е”ҜдёҖжҖ§пјҢйҒҝе…ҚеҮәзҺ°еҶІзӘҒгҖӮ
  • passwordпјҡз”ЁжҲ·зҡ„зҷ»еҪ•еҜҶз ҒпјҢдёҚиғҪдёәз©әгҖӮдёәдәҶдҝқиҜҒз”ЁжҲ·дҝЎжҒҜзҡ„е®үе…ЁпјҢжҲ‘д»¬йҖҡеёёдјҡеҜ№еҜҶз ҒиҝӣиЎҢеҠ еҜҶеӨ„зҗҶгҖӮ
  • roleпјҡз”ЁжҲ·зҡ„и§’иүІпјҢдҫӢеҰӮз®ЎзҗҶе‘ҳгҖҒжөӢиҜ•дәәе‘ҳзӯүгҖӮйҖҡиҝҮи§’иүІзҡ„еҢәеҲҶпјҢжҲ‘д»¬еҸҜд»ҘдёәдёҚеҗҢз”ЁжҲ·иөӢдәҲдёҚеҗҢзҡ„жқғйҷҗгҖӮ

еӣӣгҖҒжҸ’е…ҘжөӢиҜ•ж•°жҚ®пјҲеҸҜйҖүпјү рҹ§Ә

дёәдәҶжөӢиҜ•зҷ»еҪ•жіЁеҶҢеҠҹиғҪпјҢжҲ‘д»¬еҸҜд»ҘжҸ’е…ҘдёҖдәӣжөӢиҜ•ж•°жҚ®еҲ° users иЎЁдёӯгҖӮд»ҘдёӢжҳҜжҸ’е…ҘжөӢиҜ•ж•°жҚ®зҡ„ SQL иҜӯеҸҘпјҡ

жҸ’е…ҘжөӢиҜ•ж•°жҚ®еҗҺпјҢжҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”Ёиҝҷдәӣж•°жҚ®иҝӣиЎҢзҷ»еҪ•жіЁеҶҢеҠҹиғҪзҡ„жөӢиҜ•е•ҰрҹҳҺгҖӮ

дә”гҖҒйӘҢиҜҒж•°жҚ®еә“жҗӯе»ә вң…

еҸҜд»ҘдҪҝз”Ёд»ҘдёӢ SQL иҜӯеҸҘжҹҘиҜў users иЎЁдёӯзҡ„ж•°жҚ®пјҢйӘҢиҜҒж•°жҚ®еә“жҳҜеҗҰжҗӯе»әжҲҗеҠҹпјҡ

еҰӮжһңиғҪеӨҹжӯЈеёёжҹҘиҜўеҲ°жҸ’е…Ҙзҡ„жөӢиҜ•ж•°жҚ®пјҢиҜҙжҳҺж•°жҚ®еә“жҗӯе»әжҲҗеҠҹрҹ‘ҸгҖӮиҝҷдёҖжӯҘе°ұеғҸжҳҜжҲ‘д»¬зӣ–еҘҪжҲҝеӯҗеҗҺпјҢиҰҒжЈҖжҹҘдёҖдёӢжҲҝеӯҗжҳҜеҗҰзүўеӣәдёҖж ·гҖӮ

е…ӯгҖҒдёҺеә”з”ЁзЁӢеәҸйӣҶжҲҗ рҹӨқ
