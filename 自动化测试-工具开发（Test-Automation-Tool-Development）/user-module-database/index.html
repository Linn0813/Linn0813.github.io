  • username：用户的登录名，不能为空且必须唯一。这是用户登录时使用的名称，要保证其唯一性，避免出现冲突。
  • password：用户的登录密码，不能为空。为了保证用户信息的安全，我们通常会对密码进行加密处理。
  • role：用户的角色，例如管理员、测试人员等。通过角色的区分，我们可以为不同用户赋予不同的权限。
